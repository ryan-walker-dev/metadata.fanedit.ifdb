# metadata.fanedit.ifdb
Kodi Movie Information Scraper for the Internet Fanedit Database using a google to search (due to the limitation of IFDB's search)

Metadata agent for scraping information from the Internet Fanedit Database (IFDB)
Filename titles and search queries must be able to be found by searching IFDB via google

Install:
1. Create a zip file containing all of the files in this Repo
2. On Kodi select "Install from Zip File" and select the created Zip file
3. Choose IFDB as the source for the section
